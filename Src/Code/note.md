# How to run Flask programme

## 01 Installation

1. Python Version

    Recommend using the latest version of Python 3. Flask supports Python 3.5 and newer.

2. Install Flask

    ```python
    pip install Flask
    ```

## 02 Run 

1. Windows environment
    ```python
    set FLASK_APP=app.py

    flask run
    ```
2. Linux-like environment
    ```python
    export FLASK_APP=app.py

    flask run
    ```
